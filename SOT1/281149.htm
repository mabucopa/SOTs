Motor_Details

Smart Object content

This smart object provides a symbol representing a motor. When clicking on this symbol while runtime a screen showing details is poping up.

  • Mode can be switched between manual and automatic
  • Command buttons to start and stop the motor in manual mode
  • Speed setpoint can be set in manual mode
  • Speed setpoint in automatic mode from any mapped variable
  • Adjustable speed ramp for up and down, can be disabled
  • Motor can run in positive and negative direction
  • Actual value options for current, power and torque, can be disabled
  • Adjustable warnings and alarms for actual values
  • Simulation mode for testing including some noise for actual values
  • Trend and gant screens
  • Alarm screen
  • Chronological event list screen

Prerequisites

To use this smart object it is mandatory to have smart object template Motor_LoadAll (Version 2) imported and referenced as Motor_Global. If not the zenonLogic project LoadParameter will not compile and you can not make use of parameter loading to multiple motors.
